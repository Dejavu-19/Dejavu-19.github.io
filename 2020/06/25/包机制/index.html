包机制

为了更好的组织类，Java提供包机制用于区别类名的命名空间

包语句的语法格式：

package pkg.1[.pkg2[.pkg3…]];

一般利用公司域名倒置作为包名；www.baidu.com - com.baidu.www

导入包import

import package1[package2…].(classname|*);

.*代表通配符可以导入这个包下所有的类
